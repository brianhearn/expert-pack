## Phase Dependencies

Phases are sequential — each phase outputs prerequisites for the next:
- Phase 1 (site) → determines feasibility and constraints for Phase 2 (design)
- Phase 2 (design) → determines component specs needed for Phase 3 (selection)
- Phase 3 (selection) → permits require specific equipment lists → Phase 4
- Phase 4 (permits approved) → gate to start Phase 5 (installation)
- Phase 5 (installation complete) → gate to Phase 6 (inspection)
- Phase 6 (utility approval) → gate to Phase 7 (live operation)

## Parallel Work

Phases 3 and 4 have some overlap: you can start the interconnection application before finalizing equipment, and you can order materials while waiting for permit approval (risky but common). Phase 7 starts the moment the system is commissioned and continues indefinitely.
